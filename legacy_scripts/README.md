# Legacy implementation scripts

This folder contains the original implementation scripts. They are kept here as backups for traceability.

The repository root contains the ordered entry scripts used by `run_all.R` and `run_codeocean.R`. Each ordered script calls the corresponding implementation script in this folder with `source()`.

Do not run files in this folder directly unless you are reproducing or debugging the original implementation. Use the ordered scripts in the repository root for routine analysis.
